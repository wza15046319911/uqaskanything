How do I manage my wired internet connection?

  1. Home
  2. Information and services

About UQ Internet Management System (IMS) 

For security, UQ wired networks at UQ campuses and sites require authentication before devices can access external websites and resources.

Sometimes, you may be able to access UQ pages but not external websites. This can occur if you do not have an active Internet session. You can activate an Internet session via the UQ Agent (Windows), UQ IMS Portal (macOS) or your my.UQ Dashboard. 

If you are transitioning from Wi-Fi to a wired connection by connecting to a dock, you may need to manually activate an Internet session.

Please note: Wireless connections activate an Internet session automatically.
 

Exceptions for core UQ services

Certain services like Outlook and Teams may still be accessible without an active Internet session. These services are treated as exceptions due to their critical nature for UQ operations and can function without full external internet access.
 

Using the UQ Agent on Windows 

The UQ Agent application can be used to manage your wired Internet connection if you're using an ITS-managed Windows computer. The agent automatically activates and manages your Internet session for you if you are connected to the network at login.

You can access the UQ Agent in the system tray at the bottom right of the screen.

To activate an Internet session manually, right-click on the UQ Agent icon in the system tray and select ‘Start'.

If the UQ Agent is not available, it can be installed via Company Portal.
 

Using the UQ IMS Portal on macOS

You can use the UQ IMS Portal application to manage your wired Internet connection if you're using an ITS-managed macOS computer. The agent automatically activates and manages your Internet session for you if you are connected to the network at login.

You can access the UQ IMS Portal in the menu bar at the top right of the screen.

To activate an Internet session manually, click on the UQ IMS Portal icon in the menu bar and select ‘Force Login’.

If the UQ IMS Portal is not available, it can be installed via Self Service.
 

Using a Web Browser

If you do not have the agent/portal installed or you are using a non-managed device, you can activate and manage your wired Internet sessions through your my.UQDashboard > Internet Session (UQ login required). You can also activate an Internet session by visiting https://login.uq.edu.au
