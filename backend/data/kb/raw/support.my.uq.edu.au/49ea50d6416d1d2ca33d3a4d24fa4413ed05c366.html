How do I transfer files between my computer and the web browser virtual desktop?

  1. Home
  2. Information and services
If you are using the web browser version of the virtual desktop and need to access files on your computer, you can transfer them to and from the virtual desktop.
 

Upload files from your computer to the virtual desktop

  1. In the virtual desktop, click the side bar menu button on the left of the screen.
  2. Click the 'Open File Transfer panel' button.
  3. Select the 'Upload' tab.
  4. Drag and drop the file(s) into the window, or select 'Choose Files'.

 

Accessing uploaded files

  1. Click the 'Start' menu, then open 'File Explorer'.
  2. Open 'Documents'.
If you need to access a file from within an application:
  1. When you are in the application's Open menu, type %userprofile%\Documents into the address bar, then press Enter.
  2. If you don't see the file, click the file type drop down and select 'All Files'.

 

Download files from the virtual desktop back to your computer

  1. In the virtual desktop, click the side bar menu button on the left of the screen.
  2. Click the 'Open File Transfer panel' button.
  3. Select the 'Download' tab.
  4. With the 'Download' tab open, select the file(s) you want to transfer, then press 'Ctrl + C' on your keyboard to start the transfer.


     
  5. The file(s) will appear green. Click the download icon to save the file(s) to your computer.
